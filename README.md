## 🌍 Why AI-Era SEO?

Search is no longer just Google. Millions of users now discover content through **ChatGPT, Gemini, Perplexity, Claude, and other AI systems**. These models read, index, and summarize your content — but their rules are different from traditional search engines.

This checklist helps you:
- ✅ Stay visible in traditional search (Google, Bing)
- ✅ Get cited by AI assistants and LLM-powered search
- ✅ Future-proof your content strategy

---

## 📋 Quick Checklist

Use this as a fast-scan overview. Click any item for full details.

### 🤖 LLM Discoverability
- [ ] [`llms.txt` file created and published](sections/01-llms-txt.md)
- [ ] `llms-full.txt` with full page content available
- [ ] AI crawlers whitelisted in `robots.txt`
- [ ] Sitemap submitted and up to date

### 🏗️ Structured Data
- [ ] JSON-LD schema implemented on all key pages
- [ ] `Article`, `FAQPage`, `HowTo`, or `Product` schema where relevant
- [ ] `sameAs` property linking to authoritative sources
- [ ] Schema tested via Rich Results Test

### 💡 Content & Entity Optimization
- [ ] Author entity clearly defined (`Person` / `Organization` schema)
- [ ] E-E-A-T signals present (author bio, credentials, citations)
- [ ] Named entities are accurate and consistent across the site
- [ ] Content answers questions directly (LLMs prefer direct answers)

### 🔐 Technical SEO
- [ ] Core Web Vitals passing (LCP, CLS, FID/INP)
- [ ] HTTPS enabled
- [ ] Canonical tags set correctly
- [ ] No critical errors in Google Search Console

### 📊 Monitoring & Analytics
- [ ] Tracking AI-driven referrals (Perplexity, ChatGPT)
- [ ] Brand mentions monitored via Google Alerts / mention.com
- [ ] Regular crawl health checks
